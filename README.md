## 🎓 Overview

> **College ERP System** is a fully functional, multi-module Enterprise Resource Planning web application built for college management. It features a complete Student Dashboard with Attendance tracking, Notes, Assignments, Results, Fee Management, Faculty Panel, Placement (Recruiters), and Student Profile — all in one sleek, responsive interface with Dark/Light theme support. Built entirely with vanilla HTML, CSS & JavaScript — no frameworks, no backend.

---

## 🚀 Modules & Features

### 📊 Dashboard
- 📈 **Live Stats Cards** — Total Students, Present Today, Pending Assignments, Pending Fees
- 📉 **Attendance Overview** — Visual summary of attendance data
- 💰 **Fee Overview** — Quick snapshot of fee collection status
- 🔔 **Notifications Panel** — Assignment due, Attendance updated, Fee receipt generated
- 🖨️ **Print & PDF Export** — Export dashboard data instantly

### 📋 Attendance Module
- ➕ **Add Attendance** — Mark students as Present / Absent / Late
- 🔍 **Search** — Find student attendance records instantly
- 📝 **Attendance Table** — Name, Roll No, Class, Status, Action columns
- 💾 **Save Attendance** — Store records for the session

### 📝 Notes Module
- ➕ **Create Notes** — Add Subject, Title, Description, Type & Semester
- 🔍 **Search Notes** — Filter by subject or title
- 📄 **Notes Table** — Subject, Title, Type, Semester, Action
- 🗑️ **Delete / Edit** — Manage notes with action buttons

### 📌 Assignments Module
- ➕ **New Assignment** — Add Title, Subject, Due Date & Status
- 🔄 **Status Tracking** — Pending / Submitted / Completed
- 🔍 **Search Assignments** — Find any assignment quickly
- 📅 **Due Date Display** — Never miss a deadline

### 📊 Results Module
- 🔍 **Search Results** — Find student results by name/subject
- 📋 **Results Table** — Student, Subject, Marks, Grade, Remark, Action
- 🎯 **Grade Display** — Auto-calculated grades based on marks

### 💰 Fee Module
- ➕ **Add Fee Record** — Log total fee, paid amount, due amount
- 📊 **Fee Table** — Student, Total Fee, Paid, Due, Status columns
- ✅ **Payment Status** — Track Paid / Pending / Partial records

### 👨‍🏫 Faculty Panel
- 🎓 **Faculty Management Area** — Premium UI for faculty actions
- 📊 **Placement Stats** — Placed Students, Companies Visited, Avg Package, Open Roles

### 💼 Recruiters / Placement
- 📈 **312** Placed Students
- 🏢 **46** Companies Visited
- 💰 **6.8 LPA** Average Package

### 👤 Student Profile
- ✏️ **Edit Profile** — Name, Email, Roll No, Course, About
- 💾 **Save Profile** — Persist student information

### 🎨 UI / UX
- 🌙 **Dark / Light Theme Toggle** — Switch themes instantly
- 📱 **Fully Responsive** — Mobile, tablet & desktop ready
- ☰ **Sidebar Navigation** — Smooth slide-in menu with all modules
- 🔔 **Notification System** — Real-time alerts panel
- 🔐 **Login / Sign Up Modal** — Auth UI with clean form design
- 🖨️ **Print & PDF** — Export functionality built in

## ⚙️ How It Works

```javascript
1. Login/Signup modal handles auth UI on page load
2. Sidebar navigation switches between 8+ modules dynamically
3. Dashboard stats update based on attendance & fee records
4. Each module has Add/Search/Save/Delete functionality via DOM
5. Attendance, Notes, Assignments stored & rendered in tables
6. Fee module tracks paid/due status per student
7. Theme toggle switches CSS variables for dark/light mode
8. Notification panel shows live alerts for key events
9. Print/PDF buttons trigger browser print & jsPDF export
```
